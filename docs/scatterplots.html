2.1 Scatterplots

Given a numeric variable(s) on x and y, plot_ly() defaults to a scatterplot

subplot(
  plot_ly(mtcars, x = ~wt, y = ~mpg),
  plot_ly(mtcars, x = ~wt, y = ~mpg, color = ~disp),
  plot_ly(mtcars, x = ~wt, y = ~mpg, color = ~factor(vs))
)
